Remaining Ques 10. What is an Immediately Invoked Function in JavaScript? An Immediately Invoked Function ( known as IIFE and pronounced as IIFY) is a function that runs as soon as it is defined. (function(){ // Do something; })(); An IIFE (Immediately Invoked Function Expression) is a JavaScript function that is executed as soon as it is defined. IIFE is used to create a private scope, avoiding polluting the global namespace and providing encapsulation.
